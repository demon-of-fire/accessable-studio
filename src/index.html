User Guide

This guide explains all the features and commands available in Accessible Studio. Everything is keyboard accessible and works with NVDA.

Getting Started

  1. Use the sidebar buttons on the left to switch between Video Editor, Photo Editor, File Converter, and Assistant.
  2. In the Video Editor, click Import Media to add video or image files.
  3. Your imported files are automatically added to the timeline at the playhead position. They also appear in the Media Library for reference.
  4. Use the playback controls to preview your video.
  5. When finished, click Export Video to save your project.

File Converter

The File Converter supports converting between:

  • Video formats: MP4, AVI, MKV, MOV, WebM, WMV, GIF
  • Audio formats: MP3, WAV, OGG, FLAC, AAC
  • Image formats: PNG, JPG, WebP, BMP, GIF, TIFF
  • Document formats: PDF, Word (DOCX), Plain Text, HTML

Photo Editor

The Photo Editor lets you:

  • Open images in common formats (PNG, JPG, BMP, WebP, etc.)
  • Crop, resize, rotate, and flip images
  • Adjust brightness, contrast, saturation, and sharpness
  • Apply filter presets (Grayscale, Sepia, Vintage, Warm, Cool, Vivid, Noir)
  • Add text overlays to images
  • Save in different formats

Tips for Screen Reader Users

  • All buttons have descriptive labels that your screen reader will read.
  • Use Tab and Shift+Tab to move between controls.
  • Sliders (like volume and filters) can be adjusted with Left/Right arrow keys.
  • The status bar at the bottom announces important changes.
  • Dialog boxes trap focus so you stay inside them until you close with Escape or the Close button.
  • The Assistant section lets you type commands instead of clicking buttons, which may be faster.

About Accessible Studio

Accessible Studio is a fully accessible video editor, photo editor, and file converter designed to work with NVDA and other screen readers.

Every button and control is properly labeled for screen reader users. All features can be accessed via keyboard.
