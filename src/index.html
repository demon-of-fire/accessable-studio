User Guide

Getting Started

  1. Use the Mode sidebar on the left to switch between Video Editor, Photo Editor, File Converter, User Guide, and Settings.
  2. In each mode, use the dropdown menus at the top (File, Edit, Effects, Insert, Tools) to access all editing options.
  3. In the Video Editor, open File > Import Media (or press Ctrl+I) to add video or image files. Use File > Import Audio for music and sound effects.
  4. Each imported file prompts you to choose where on the timeline it goes (start of track, playhead position, end of track, etc.).
  5. Edit your clips using the menus or type commands in the assistant bar at the bottom of the screen.
  6. Use the playback controls to preview your video. Press Space to play/pause.
  7. Click Save Project to save your work. Your projects appear in the Projects panel on the right side, where you can reopen them anytime.
  8. When finished, click Export Video (or press Ctrl+E) to render your final video.
  9. The assistant bar at the bottom accepts natural language commands on any tab. Just type and press Enter — responses appear inline above the input.

Assistant Commands

The assistant bar sits at the very bottom of every tab. Type a command and press Enter — the response appears inline above the input. Press Ctrl+B to focus the assistant from anywhere.

Gemini AI (Recommended)

When a Gemini API key is set (in Settings), the assistant is powered by Google Gemini 2.5 Pro. It can see your photo, knows every clip on your timeline, and makes real edits — not just suggestions. It understands complex, multi-step requests like "select the second clip, trim 3 seconds from the start, and apply a cinematic filter".

Without a Gemini key, the assistant still works using built-in keyword matching for all standard commands listed below.

File Converter

The File Converter supports converting between:

  • Video formats: MP4, AVI, MKV, MOV, WebM, WMV, GIF, FLV, MPEG, M4V, TS (MPEG-TS).
  • Audio formats: MP3, WAV, OGG, FLAC, AAC, WMA, M4A, OPUS.
  • Image formats: PNG, JPG, WebP, BMP, GIF, TIFF, ICO, SVG, AVIF, EPS.
  • Document formats: PDF, DOCX, TXT, HTML, RTF, Markdown, CSV, EPUB, ODT. Select your document, pick an output format, and click Convert.

Video and audio conversion requires FFmpeg to be installed. Document conversion works out of the box.

Photo Editor

The Photo Editor lets you open images, crop, resize, rotate, flip, adjust brightness and contrast, apply filter presets, add text overlays, remove or blur backgrounds, insert images, and save in different formats (PNG, JPG, WebP, BMP, GIF, TIFF).

Tips for Screen Reader Users

  • All buttons have descriptive labels that your screen reader will read.
  • Use Tab and Shift plus Tab to move between controls.
  • Sliders like volume and filters can be adjusted with Left and Right arrow keys.
  • The status bar at the bottom announces important changes.
  • Dialog boxes trap focus so you stay inside them until you close with Escape or the Close button.
  • The assistant bar at the bottom lets you type commands anytime. Responses appear inline above the input. Click the X button or type a new command to dismiss.
  • The clip list at the bottom shows all clips on the left and detached clips on the right. Select a clip to see its properties, including rename, fade in/out, and layer controls.
  • Markers on the timeline are accessible via chatbot commands: "next marker", "previous marker", "list markers".
  • All new feature dialogs (Chroma Key, Keyframe, Speed Ramp, Ken Burns, PIP, Title Templates, Color Correction) have proper aria-labels, aria-modal, and focus trapping for full screen reader support.
  • Clip indicators (chroma key, ken burns, keyframes, speed ramp, PIP, freeze frame) are shown as text badges in the clip list items for screen reader visibility.
